# Group Policies

## Goal
Create and manage Group Policy Objects to automate tasks and enforce permissions.

## Tasks
1. Open Group Policy Management
2. Create a new GPO
3. Assign to appropriate OU
4. Configure security settings, login scripts, or folder redirection
5. Test GPO effects on client VM
