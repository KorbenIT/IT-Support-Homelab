# Client Configuration

## Goal
Configure Windows 10/11 clients to connect to domain and test policies.

## Tasks
1. Set client IP (DHCP or static as needed)
2. Join client to domain
3. Log in with domain user
4. Verify GPO application
5. Troubleshoot access issues
