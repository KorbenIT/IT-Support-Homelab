# Windows Server & Active Directory Lab

## Objective
Gain hands-on experience with Windows Server, Active Directory, Group Policy, and client-server interactions for IT support roles.

## Skills Demonstrated
- Windows Server installation and configuration
- Active Directory Domain Services (AD DS)
- User and group management
- Group Policy Objects (GPO) creation and deployment
- Joining client VMs to the domain
- Realistic troubleshooting scenarios

## Lab Structure
- server-setup.md → Windows Server VM setup
- active-directory.md → AD DS installation and configuration
- group-policies.md → GPO creation and management
- client-configuration.md → Domain join and client setup
- it-support-scenarios.md → Troubleshooting tasks
- screenshots/ → Organized visual documentation per task
