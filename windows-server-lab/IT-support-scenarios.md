# IT Support Scenarios

## Goal
Document realistic troubleshooting scenarios for Windows Server and clients.

## Examples
- Domain login failure → check DNS and client network
- Permission denied on shared folder → check NTFS and GPO
- Script or GPO not applying → check OU, replication, and policy refresh
