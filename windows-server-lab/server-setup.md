# Windows Server Setup

## Goal
Install and configure Windows Server 2019/2022 VM for domain controller functionality.

## Steps
1. Download Windows Server ISO
2. Create VM in VirtualBox/VMware
   - RAM: 4-8GB, CPU: 2 cores, Disk: 60GB
3. Install Windows Server
4. Configure static IP and hostname
5. Install updates and take a snapshot
