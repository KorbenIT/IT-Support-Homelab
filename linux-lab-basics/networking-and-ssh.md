# Linux Networking and SSH

## Goal
Set up networking and SSH access to communicate with Windows VM.

## Tasks
1. Assign static IP if needed
2. Test connectivity: ping Windows VM / host
3. Install and test SSH server: `sudo apt install openssh-server`
4. SSH from Windows to Ubuntu
