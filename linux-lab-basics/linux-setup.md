# Linux Setup

## Goal
Install Ubuntu LTS and configure the VM environment.

## Steps
1. Download Ubuntu ISO
2. Create VM in VirtualBox / VMware
3. Allocate RAM: 2-4GB, CPU: 2 cores, Disk: 30GB
4. Install Ubuntu LTS
5. Update system using `sudo apt update && sudo apt upgrade`
6. Take snapshot after installation
