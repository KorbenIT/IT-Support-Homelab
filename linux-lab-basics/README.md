# Linux Basics Lab

## Objective
Gain hands-on experience with Linux system administration and command-line tools commonly used in IT support roles.

## Skills Demonstrated
- Ubuntu LTS installation and configuration
- File system navigation
- Networking and SSH access
- Package management using apt
- Basic shell scripting and automation

## Lab Structure
- linux-setup.md → Installation and VM setup
- networking-and-ssh.md → Networking configuration and testing
- package-management.md → Software installation and updates
- shell-scripting-basics.md → Automation using scripts
- screenshots/ → Visual documentation per task
