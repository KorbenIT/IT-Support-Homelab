# Linux Package Management

## Goal
Practice installing, updating, and troubleshooting software using `apt`.

## Tasks
1. Update package lists: `sudo apt update`
2. Upgrade packages: `sudo apt upgrade`
3. Install software: e.g., `sudo apt install vim git curl`
4. Document any installation errors and resolutions
