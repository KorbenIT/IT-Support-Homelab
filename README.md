# My IT Support Home Lab

Hey there! This repo is my personal IT home lab, where I’ve been building hands-on experience with Windows, Linux, networking and server administration. My goal was to simulate real-world IT support scenarios, learn by doing, and document everything in a way I can share with potential employers.

---

## What I’ve Learned / Skills Highlighted
- Setting up and troubleshooting Windows 10/11 VMs  
- Navigating and managing Ubuntu Linux systems  
- Networking basics and firewall configuration using pfSense  
- Windows Server administration, Active Directory, and Group Policy  
- Real-world IT troubleshooting, system recovery, and documentation  

---

## Labs Overview

### 1. Windows Fundamentals Lab
- Installed and configured Windows 10/11 VMs  
- Worked with NTFS permissions, shared folders, and file system management  
- Practiced basic networking setup and troubleshooting  
- Ran realistic troubleshooting scenarios  

### 2. Linux Basics Lab
- Installed Ubuntu LTS and got comfortable navigating the system  
- Configured networking and SSH between Linux and Windows VMs  
- Managed software with `apt` and handled update/installation issues  
- Wrote simple shell scripts and experimented with automation  

### 3. Networking & pfSense Lab
- Built a small virtual office network connecting multiple VMs  
- Configured pfSense as the router/firewall  
- Set up DHCP, DNS, and firewall rules  
- Tested connectivity and troubleshot network issues  

### 4. Windows Server & Active Directory Lab
- Installed Windows Server 2019/2022  
- Set up Active Directory with users, groups, and OUs  
- Created and tested Group Policy Objects (GPOs)  
- Joined Windows client VMs to the domain  
- Documented IT support scenarios and fixed problems along the way
