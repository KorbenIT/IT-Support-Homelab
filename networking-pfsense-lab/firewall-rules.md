# Firewall Rules

## Goal
Create and test firewall rules to control traffic between VMs and internet.

## Tasks
1. Create basic allow/deny rules
2. Test ping, SSH, HTTP/HTTPS connectivity
3. Configure DHCP for LAN
4. Configure DNS forwarding
5. Test cross-VM communication
