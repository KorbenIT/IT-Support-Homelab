# Networking & Firewall Lab

## Objective
Simulate a small office network using pfSense to manage routing, firewall rules, and IP address allocation.

## Skills Demonstrated
- Network design and topology planning
- pfSense installation and configuration
- DHCP and DNS setup
- Firewall rule creation and troubleshooting
- VM-to-VM communication

## Lab Structure
- network-design.md → Diagrams and network plan
- pfsense-setup.md → pfSense installation and VM configuration
- firewall-rules.md → Create and test rules
- troubleshooting.md → Record problems and resolutions
- screenshots/ → Organized visual documentation
