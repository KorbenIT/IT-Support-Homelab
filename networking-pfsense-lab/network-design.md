# Network Design

## Goal
Plan a small office network connecting multiple VMs (Windows + Linux) through pfSense.

## Tasks
1. Draw network topology diagram
   - Include: pfSense, LAN, client VMs, DHCP/DNS
2. Note IP ranges and subnets
3. Plan firewall rules and NAT if needed
