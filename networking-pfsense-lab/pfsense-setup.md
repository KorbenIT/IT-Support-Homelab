# pfSense Setup

## Goal
Install and configure pfSense VM as the network router/firewall.

## Steps
1. Download pfSense ISO
2. Create VM in VirtualBox/VMware
   - RAM: 2-4GB, CPU: 2 cores, Disk: 10-20GB
3. Install pfSense
4. Configure WAN (NAT or Bridged)
5. Configure LAN (internal network connecting all VMs)
6. Verify connectivity from client VMs
