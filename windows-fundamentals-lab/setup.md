# Windows 10 VM Setup

## Goal
Install and configure a Windows 10 Pro virtual machine.

## Steps
1. Download Windows 10 ISO.
2. Create VM in VirtualBox or VMware.
3. Allocate RAM: 4GB, CPU: 2 cores, Disk: 50GB.
4. Install Windows 10 Pro.
5. Install updates and VM Guest Additions.
6. Take a snapshot after clean installation.
